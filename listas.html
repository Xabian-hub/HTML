Ejemplo de Listas en HTML

Lista Ordenada

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista Sin Orden

  • Elemento A
  • Elemento B
  • Elemento C

Diferencia entre lista ordenada y desordenada

Una lista ordenada (<ol>) muestra sus elementos en un orden numérico o alfabético, lo que implica una secuencia específica. Cada elemento está numerado o listado en orden.Un ejemplo muy basico es una receta.

Una lista sin orden (<ul>) no tiene un orden numérico ni alfabético; simplemente muestra los elementos como puntos o viñetas. El orden de los elementos no implica ninguna jerarquía o secuencia.Un ejemplo muy basico es la lista de la compra.
