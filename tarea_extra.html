Tabla de ejemplos

En esta tabla hay informacion sobre todas las etiquetas utilizadas en la web

Etiqueta Descripción
<h1> Encabezado de nivel 1
<img> Etiqueta para insertar imágenes
<a> Enlace para vincular a otras páginas o recursos
<title> Define el título de la página (en la pestaña del navegador)
<header> Sección de cabecera de la página
<main> Contenedor principal del contenido de la página
<section> Sección de contenido dentro de la página
<h2> Encabezado de nivel 2
<p> Parrafo de texto
<style> Define estilos CSS dentro del HTML (esto lo veremos despues pero ya que estaba lo añadia)
<div> Contenedor genérico para estructurar el contenido
<footer> Pie de página de la web
<ol> Lista ordenada (numerada)
<li> Elemento de una lista (desordenada u ordenada)
<ul> Lista desordenada (con viñetas)
<strong> Texto en negrita (enfatizado)
<code> Define un fragmento de código (para que el codigo no ejecute la accion, sino que plasme el texto literal)
<center> Centra el contenido dentro de su contenedor
<table> Crea una tabla
<tr> Define una fila dentro de la tabla
<th> Define una celda de encabezado en la tabla
<td> Define una celda dentro de una fila de la tabla
